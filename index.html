简介

BWM出品的用于快速简便地创建UIScrollView和UIPageControl的图片展示封面 支持循环滚动，动画，异步加载图片，等功能。

如何使用

快速创建BWMCoverView models是一个包含图片地址的数组 placeholderImageNamed为图片加载前的本地占位图片名

BWMCoverView *coverView = [BWMCoverView coverViewWithModels:realArray andFrame:CGRectMake(10, 50, 300, 400) andPlaceholderImageNamed:@"defaultImage" andClickdCallBlock:^(NSInteger index) { NSLog(@"你选择了第%d个图片", index); }];

// 请打开下面的东西逐个调试 // [coverView setAutoPlayWithDelay:3.0]; // 设置自动播放 // [coverView stopAutoPlayWithBOOL:YES]; // 停止自动播放 // [coverView stopAutoPlayWithBOOL:NO]; // 恢复开启自动播放 // [coverView setAnimationOption:UIViewAnimationOptionTransitionCurlUp]; // 设置切换动画

[self.view addSubview:coverView];
